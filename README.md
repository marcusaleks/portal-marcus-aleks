# portal-marcus-aleks

Portal público de mercado financeiro brasileiro e ferramentas de análise desenvolvidas pela MAD Developers.

## Stack

- **Frontend/Deploy:** Next.js + Vercel (legado — migração para Cloudflare planejada)
- **Estilo:** Tailwind CSS v4
- **Dados:** BRAPI (cotações), BCB API (SELIC), AwesomeAPI (USD/BRL)

## Funcionalidades

- Dashboard público: IBOVESPA, USD/BRL, SELIC Efetiva, calendário COPOM
- Banner rotativo com 18 ativos em tempo real
- Área restrita: SIFAZ — Sistema de Inteligência Fazendária
- Hub OSINT com ferramentas de investigação

## Governança

Projeto desenvolvido em conformidade com a **Lei de Operação MAD v1.0** (STACK-MAI26).
